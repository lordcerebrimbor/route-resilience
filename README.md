# Route Resilience

Route Resilience is our team's project for analysing the resilience of urban road networks using graph theory and AI.

The project aims to move beyond road extraction by identifying critical roads, simulating disaster scenarios, and providing transparent decision support for urban planners.

## Current Repository

This repository contains the interactive dashboard used to visualize our graph intelligence and network resilience analysis.

## Features

- Interactive network visualization
- Degree, Betweenness, Closeness and Eigenvector centrality
- Critical road ranking
- Road blockage simulation
- Network resilience metrics
- Resilience gap analysis
